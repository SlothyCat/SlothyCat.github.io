# Introduction
Apparently, AWS also helps with training AI and ML models and provides services for Data Analytics. 🤔

### AI/ML on AWS
ML is used internally at AWS, but it also recognises that clients could leverage its models to build something more customised for their own use cases. 

| Solution Type                   | Description                                                                                                                                     |
| ------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| AI                              | **pre-built models** that are already trained to perform specific functions                                                                     |
| ML                              | a **more customised approach** with Amazon SageMaker AI where you build, train, and deploy your own ML models with fully managed infrastructure |
| ML Framework and Infrastructure | **completely custom approach** to building models using purpose-built chips that integrate with popular ML frameworks                           |

### Tier 1: Pre-Built AWS AI Services

### Tier 2: ML Services
**More customised approach** for customers who want a bit more control over their ML solutions without having to manage infrastructure.

#### Amazon SageMaker AI
With this fully managed service, you can build, train, and deploy your own ML models without worrying about infrastructure. You can track model training, visualisation, debugging and monitoring workflow all under it. It also has access to pre-trained models that you can deploy.

As the workflow is repeatable, it helps with transparency and auditability.


### Tier 3: ML frameworks and infrastructure
**Highly specialised needs** that require complete control over the ML training process. Clients can DIY using the framework and infrastructure that AWS has to offer.

### Generative AI on AWS
Gen AI is a subset of deep learning with extremely large ML models known as **foundation models** (FMs). FMs are trained on vast collections of data, and FMs can be adapted to multiple tasks.

#### Amazon SageMaker JumpStart
Offers a library of pre-built ML solutions across various domains such as computer vision, NLP, and tabular data. These pre-trained models can be fine-tuned to suit your specific needs and deployed with just a few clicks.

**Use Cases:** Rapid ML Model Deployment, Custom fine-tuned solutions, ML experiments and Prototypes


#### Amazon Bedrock
**Fully managed service** that was specifically designed for working with large foundation models and building generative AI applications.

**Use Cases:** Enterprise-Grade Gen-AI, Multimodal content generation, Advanced Conversational AI


#### Amazon Q products
A generative AI assistant that can help companies streamline processes, make decisions faster, and improve employee productivity.

There are 2 products:
**Q Business**: answer pressing questions, help solve problems, and take actions using the data and expertise found in your company's information repositories.
**Q Developer**: provides code recommendations to accelerate development for coding languages

### Data Analytics on AWS
Both AI/ML and traditional data analytics need clean and accessible data in a format that's usable by analytics tools and AI algorithms.

#### Extract Transform Load (ETL)
1. _Extract_ the data from various sources and store it
2. _Transform_ it into a consistent, usable format for downstream tools to consume
3. _Load_ it into a destination system, like a data warehouse or analytics platform

Data pipelines are automated assembly lines used to make the ETL process efficient and repeatable.

And then we can have Data Analytics based on this data that has been transformed to be usable!

#### Data ingestion services
Data ingestion involves moving data from source systems into your chosen storage solution.

**Amazon Kinesis Data Streams**: real-time ingestion of TBs of data, serverless and provides automatic provisioning and scaling.
**Amazon Data Firehose**: near real-time, fully managed

#### Data storage services
Data can be either stored as a flexible data lake or an organised data warehouse

**Amazon S3**: The data lake, with any amount of unstructured or structured data
**Amazon Redshift**: a fully managed data warehouse service that can store petabytes of structured or semistructured data.

#### Data cataloguing services
Provides an inventory of your organisation's data, the AWS solution is  AWS Glue Data Catalogue.

#### Data processing services
Clean and transform your data so it's ready to be analysed.

**AWS Glue**: a fully managed ETL service that makes data preparation simpler, faster, and cost-effective.
**Amazon EMR**: ideal for large-scale data processing and organisations with existing big data expertise. It automatically handles infrastructure provisioning, cluster management, and scaling.

#### Data analysis and visualisation services
help you to develop important insights about your data.

| Service                       | Description                                                                                                                                                   |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Amazon Athena**             | run SQL queries to analyse data in relational, nonrelational, object, and custom data sources                                                                 |
| **Amazon Redshift**           | fully managed data warehouse solution                                                                                                                         |
| **Amazon QuickSight**         | both technical and non-technical users can quickly create modern interactive dashboards and reports from various data sources without managing infrastructure |
| **Amazon OpenSearch Service** | search for relevant content through precise keyword matching or natural language queries                                                                      |
